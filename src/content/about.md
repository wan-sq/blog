## 关于这个博客

这里主要收我的**研究笔记**：读过的文献、整理过的方法、踩过的坑，以及每周的一点进展。

写下来有两个用处：一是逼自己把模糊的理解说清楚，二是半年后回头看，还能想起当时为什么这么想。

### 你会在这里看到

- **文献管理**：我是 Zotero 的深度用户，会记一些整理和检索的小技巧
- **阅读与方法**：怎么读一篇论文、怎么做笔记、怎么把输入变成输出
- **研究周记**：每周的进展、卡点和下一步
- **实验室之外**：下班后的半小时，生活里的小事

### 我的工具链

| 用途 | 工具 |
| --- | --- |
| 文献管理 | Zotero 7（本地库 + 浏览器连接器） |
| 笔记 | 文献批注导出 + Markdown |
| 写作 | 这个博客（Vite + Markdown） |
| 检索 | Zotero 高级搜索 + 全文索引 |

### 怎么联系我

如果某篇笔记帮到了你，或者你有更好的文献工作流，欢迎写信告诉我。不一定回得快，但都会认真看。

> 管理文献不是为了收藏，而是为了下次能把它找回来。
